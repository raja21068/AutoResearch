# Citation Validation

## Summary
- **Total citations**: 20
- **All verified**: Yes
- **Validation method**: Manual check of DOI/arXiv links + metadata match

## Files
- `citation_validation.json` — full validation with DOIs, arXiv IDs, validator
- `citation_validation.csv` — tabular format

## Protocol
Each citation was independently verified by one evaluator:
1. Check arXiv/DOI link resolves
2. Verify title, authors, venue, year match
3. Confirm paper is peer-reviewed or reputable preprint
